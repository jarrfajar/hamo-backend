# Hamo API
## Installation
1. Clone the repo

   ```bash
   git clone https://github.com/jarrfajar/hamo-backend
   ```
2. Install NPM packages

   ```npm
   npm install
   ```
3. Install Composser

   ```composer
   composer install
   or
   composer update
   ```

4. Database

   ```laravel
   create a database with the name hamo
   and
   import database
   ```
